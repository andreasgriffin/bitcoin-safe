New Features

Improvements

- Descriptor UI: Improved clarity in the descriptor interface for importing formats, making the import process more intuitive
- Added import formats: Allow import of descriptors without derivation path (Sparrow format)
- Multisig Wallet Export: Multisig wallet exports now revert to the legacy (Coldcard) format, even for Q, ensuring the wallet name is included.
- Quick Receive Scrollbar (Mac): Resolved scrollbar display issues in the Quick Receive view on macOS.
- fixed install issue with the deb file
